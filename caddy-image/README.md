# Custom Caddy image builder

This is builder used to generate custom Caddy binary used in VPS deployment.
Included modules are listed below:
- https://github.com/greenpau/caddy-security
- https://github.com/caddy-dns/cloudflare
- https://github.com/mholt/caddy-l4
- https://github.com/WeidiDeng/caddy-cloudflare-ip
- https://go.akpain.net/caddy-tailscale-auth
